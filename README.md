# Tracker

**О проекте:**

Проект реализует приложение трекер заявок.

**Используемые технологии:**

- Java Core
- PostgreSQL
- JDBC, Hibernate
- JUnit, Mockito
- Apache Maven

**Реализованные возможности:**

- Добавление заявки.    
- Редактирование заявки.    
- Поиск всех заявок, поиск заявок по имени, поиск заявки по ее идентификатору.    
- Удаление заявки.
